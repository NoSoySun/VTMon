# VTMon

A Minecraft mod for VTMon.

## Description

This is a Fabric mod for Minecraft that adds VTubers converted into Pokémon for Minecraft Cobblemon.

## Installation

1. Ensure you have Minecraft and Fabric installed.
2. Download the mod JAR from releases.
3. Place the JAR in your `mods` folder.
4. Launch Minecraft with Fabric.

## Building

To build the mod:

```bash
./gradlew build
```
